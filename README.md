# Name

ダンスペアズは、社交ダンス(競技ダンス)のリーダー・パートナーを見つけるアプリです。

# Features

従来の社交ダンスリーダーパートナー掲示板は、検索や絞り込みの機能には長けているものの、それ故の使いづらさも否めません。
また、ページがごちゃごちゃしていて使いにくいと感じます。

ダンスペアズは、直感的に・簡単に・素早くダンスパートナーを見つけることができます。

# Requirement

* ruby 2.6.3p62 [x86_64-linux]
* Rails 6.0.3
